# Old Westbury Catalyst: Zero-Knowledge Masterclass

Welcome to the **Old Westbury Catalyst Masterclass**. This project was built from scratch, and this manual will teach you how to build it, too. 

You do **not** need to be a programmer to start this. We will start with the assumption that you are using a computer for the first time in a development context.

## How to use this manual
Each part contains:
1. **The Concept:** A plain-English explanation of *what* we are building and *why*.
2. **The Tools:** What to install and how.
3. **The How-To:** Step-by-step commands or coding instructions.
4. **The Why:** A breakdown of the code so you understand what it actually does.
